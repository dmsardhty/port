Nepal Earthquake : Damage Prediction of Destructed Building & Prevention Suggests

This is my second project that i got when joining data science bootcamp. The dataset was retrived from drivendata.org. This project was ran in groups consists of four people to gain a new insights based on defined business problem. Overall work that we conduct in this project are:
